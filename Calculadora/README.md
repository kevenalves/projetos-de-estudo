# Calculadora JavaScript

![Calculadora](image/calculadora.jpeg)

# Descrição
Calculadora desenvolvida utilizando estrutura de projeto MVC e orientação a objeto.

## Funcionalidades

  - Click com som ou não
  - Copiar e colar valor do display
  - Autonomia via teclado

## 🏃 Como executar o projeto

  - Execute o arquivo index.html

## 📚 Principais aprendizados
  -  Estrutura de projeto MVC
  - Programação Orientada a objetos
  - Manipulação do Dom
  - Criação e utilização de métodos
  - Manipulação de áudio
  - Manipulação de Data e Hora
  - Manipulação de array e conversão de dados
